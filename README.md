# 💼 Jobby App

A full-featured job search application built using React.  
Users can log in, search jobs, apply filters, and view detailed job information.

---

## 🚀 Features

- 🔐 User Authentication (Login with JWT)
- 🏠 Home Page with Navigation
- 🔎 Job Search with Filters
  - Employment Type
  - Salary Range
  - Search Input
- 📄 Job Details Page
- 🔁 Retry on API Failure
- 🚫 No Jobs Found View
- 🔒 Protected Routes
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- JavaScript (ES6+)
- CSS3
- REST APIs
- js-cookie
